Software

A list of software:
  • Web - For managing users and computers
  • Client - The high level client that connects to the kernel module and server
  • Kernel module - The low level management add-on
  • Server - Where all user data is stored, program settings, etc.
All of these pieces of software tie together to create an integrated experience where computers and the user can both talk to the server but also the administrators behind it.
